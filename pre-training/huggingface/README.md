# Hugging Face Pre-Training

This folder mirrors a self-supervised training loop using Transformers, with a focus on using standard tooling and understanding the defaults.

## What Is Implemented
- Dataset loading and preprocessing with Hugging Face Datasets.
- Tokenizer and model setup for causal language modeling.
- Training loop configuration using Transformers utilities.

## Notebooks
- `Self_Supervised_Learning.ipynb`: end-to-end self-supervised training run and basic evaluation.
